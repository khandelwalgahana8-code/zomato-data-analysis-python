

# Zomato Data Analysis Using Python

## Overview

This project analyzes Zomato restaurant data using Python.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Key Insights

* Dining restaurants are the most common.
* Empire Restaurant received the highest votes.
* Most restaurants do not accept online orders.
* Most ratings fall between 3.5 and 4.0.
* Couples prefer restaurants costing around ₹300 for two people.

## Visualizations

* Restaurant Type Analysis
* Ratings Distribution
* Online Order Analysis
* Cost Analysis
* Heatmap
* Box Plot
